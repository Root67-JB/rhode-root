# Sales Assistant — public website

Static one-page site served via GitHub Pages. Used to satisfy Twilio A2P
10DLC business profile requirements: a publicly accessible URL describing
the business, the SMS use case, the opt-in flow, and contact info.

Edit `index.html` to update.

## Deploy

1. Push this repo to GitHub (public).
2. Repo Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / root.
3. Wait ~1-2 minutes for GitHub Pages to provision.
4. Site URL appears at top of the Pages settings page, format:
   `https://<github-username>.github.io/<repo-name>/`
