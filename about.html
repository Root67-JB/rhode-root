About

Rhode Root is a sole-proprietor software developer based in the United States, building custom productivity software for individual professionals. The current focus is voice-first iOS applications that help working professionals manage their daily workflow hands-free.

Current product: Sales Assistant

Sales Assistant is a voice-first iOS application built and operated by Rhode Root for a single end user (a salesperson at a luxury automotive dealership). The salesperson speaks naturally to the app to look up customer information, log interactions, schedule follow-ups, and receive timely reminders about pending tasks.

  • Voice conversation: The end user talks to the app via Apple-secured voice channels. The app understands sales-specific context and performs actions on their behalf.
  • Customer + inventory lookup: Voice queries surface customer history, vehicle inventory, prior conversations, and scheduled appointments.
  • Task management: Tasks created during a voice session are tracked in a personal database tied to the end user's authenticated account.
  • SMS reminders: When a task becomes due, the system sends a text-message reminder to the end user's own verified phone number so they don't miss follow-ups while away from the app.

Who Rhode Root serves

Current operations consist of a single deployed instance serving one authenticated end user. The application does not send messages to customers, prospects, or third parties. All SMS traffic is account-internal — reminders go only to the end user's own pre-verified phone number, which is configured during initial app setup.

Privacy & data handling

  • End-user data is stored encrypted at rest in a managed Postgres database.
  • Personally identifiable information is column-encrypted using AES-256 with a separately-managed key.
  • Voice sessions are processed via Apple-authenticated channels; no third-party voice training or analytics.
  • SMS reminders contain only task titles and (optionally) a customer first name; no financial, medical, or other sensitive data is ever transmitted over SMS.

SMS message types

Rhode Root sends one type of SMS message: task reminders from the Sales Assistant application to the end user's own phone number. Example message bodies:

Task due: Follow up with John Smith

Task due: Send proposal to Mary Lee

Task due: Confirm Wednesday appointment

Recipients can opt out at any time by replying STOP to any message, or by removing the underlying task from their task list within the application.

SMS opt-in flow

The end user opts in to SMS reminders during initial setup of the Sales Assistant iOS application. The opt-in flow:

  1. End user installs the iOS application from Apple TestFlight (private distribution).
  2. End user signs in with their Apple ID and is authenticated against an allowlist.
  3. End user enters their personal phone number in app settings; the number is then pre-verified in the Twilio sender account before any messages can be sent.
  4. SMS reminders are generated only from tasks the end user has personally created in the app via voice or manual entry.
  5. Recipients can opt out by replying STOP, or by removing the task from their task list.
